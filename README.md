# ExploreGuapi

Guia digital turistico de **Guapimirim, RJ**.

Slogan: **"Guapimirim tem muito mais do que voce imagina"**.

---

## Sobre

Projeto web criado para apresentar Guapimirim como destino de natureza, aventura, cultura e gastronomia, com foco em experiencia visual moderna e responsiva.

Este repositorio esta publico **apenas para demonstracao de portifolio**.

---

## Funcionalidades

- Home com destaques da cidade
- Paginas: atracoes, roteiros, eventos, onde comer, onde ficar, guias, mapa e anuncie
- Mapa interativo por categorias
- Filtros de conteudo
- Formulario de parceiros
- SEO base (metadata, Open Graph, sitemap e robots)
- Pagina 404 personalizada

---

## Stack

- Next.js 14 + App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React Leaflet
- PostgreSQL + Prisma
